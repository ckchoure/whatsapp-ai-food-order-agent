# WhatsApp AI Food Ordering Agent (n8n + Google Gemini)

This project is an automated WhatsApp-based food ordering system built using n8n, Google Gemini, and Google Sheets. It receives customer messages, handles menu FAQs, takes orders, checks inventory, and sends confirmation with an auto-generated order ID.
